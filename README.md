# Alpine Nature Trails Map

一个展示阿尔卑斯山脉地区自然徒步路线的交互式地图应用。

## 功能特点

- 🗺️ 交互式地图展示，支持多个地图源切换
- 📍 多个地区的徒步路线标记（不同颜色区分）
- 📋 侧边栏路线列表，快速浏览所有路线
- 🔍 点击标记或列表项查看详细信息
- 📱 响应式设计，支持移动端访问

## 覆盖地区

### 斯洛文尼亚 (Slovenia) - 紫色标记
- 10条精选路线，包括文特加峡谷、布莱德湖、特里格拉夫国家公园等

### 意大利北部 (Northern Italy) - 橙色标记
- 13条路线，包括科莫湖、加尔达湖、马焦雷湖、奥尔塔湖等

### 多洛米蒂 (Dolomites) - 绿色标记
- 13条路线，包括三峰山、布拉耶斯湖、塞切达山脊、卡雷扎湖等

### 瑞士 (Switzerland) - 红色标记
- 15条路线，包括马特洪峰、少女峰、阿莱奇冰川、劳特布龙嫩谷等

### 奥地利 (Austria) - 深绿色标记
- 15条路线，包括哈尔施塔特、格绍湖、大格洛克纳山、因斯布鲁克等

### 法国 (France) - 蓝色标记
- 12条路线，包括勃朗峰、安纳西湖、瓦努瓦斯国家公园、霞慕尼等

### 德国 (Germany) - 黄色标记 ✨ 新增
- 13条路线，包括：
  - **Zugspitze Summit Area（楚格峰）** - 德国最高峰，可缆车直达
  - **Partnach Gorge（帕特纳赫峡谷）** - 狭窄峡谷与隧道式步道
  - **Mittenwald – Karwendel Panorama Trail** - 从彩绘小镇进入 Karwendel 山脉
  - **Eibsee Loop（艾布湖环湖）** - 正对楚格峰北壁的清澈湖泊
  - **Watzmann East Face View Trail（瓦茨曼山）** - 德国最具存在感的山体之一
  - **Königssee – Obersee Walk（国王湖）** - 峡谷型湖泊，需乘船进入
  - **Alpsee & Neuschwanstein Foothills** - 新天鹅堡脚下的湖泊与森林步道
  - **Berchtesgaden National Park Trail** - 德国唯一的阿尔卑斯国家公园

## 使用方法

1. 打开 `index.html` 文件
2. 地图会自动加载并显示所有路线标记
3. 点击地图上的标记或侧边栏的路线项查看详细信息
4. 使用右上角的菜单按钮打开/关闭路线列表

## 技术栈

- **Leaflet.js** - 地图库
- **纯 JavaScript** - 无框架依赖
- **CSS3** - 现代化样式设计

## 路线信息格式

每条路线包含以下信息：
- **名称** (name): 路线名称
- **坐标** (lat, lng): 地理坐标
- **距离** (distance): 路线长度
- **时长** (duration): 预计徒步时间
- **最佳季节** (bestSeason): 推荐访问时间
- **地区** (region): 所属地区（用于标记颜色）
- **描述** (description): 详细描述
